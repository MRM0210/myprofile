Project Idea:


Overview:

The project that I have decided on is a smartphone and desktop learning web application for primary school aged students and parents. The Application content will be based on the curriculum of each school year, and broken down into all the main subjects that students undertake during each year. Each subject will have quizzes that will test the child’s knowledge and report back on areas they need to improve on. Each student progress will be ranked against all other users in the system for each school year and subject. All information will also be accessible by registered parents. There will also be a discussion forum for the parents to communicate with other parent with children in the same school year.


Motivation:

The challenge of getting my two primary school aged children to complete their homework each night is the motivating factor behind this idea, to say it’s a challenge is a major understatement!!! The idea of having a fun interactive application for my children to study relevant content at the click of a button while receiving instant feedback on their work is the driving force for this project. The Idea of a platform to communicate with other parents about my child and other children’s progress is also very appealing. I am also highly motivated by the challenge of programming an application like this using and learning the different languages and technologies required to complete this task.


Description:

The application will start by asking the user to either register as a parent or student. The registration process for a student will ask the student for their country, state, name, age, gender and school year. The registration for a parent will ask for name, how many primary school aged children they have and weather they would like to link their account to a child’s account. The data provided will then be stored in a user data.

Once the user is logged in as a student a menu will appear with the following options:


  • Take a quiz
  • View learning materials
  • View profile

The “Take a quiz” option will proceed to ask the student subject and topic within the subject they would like for the quiz. After all the input is taken the application will proceed with a 20 question quiz on the subject topic and difficulty level of the school year selected on their profile, all question will be relevant to the school curriculum of the state of the student. All results for each subject will be stored in a database and the student will be notified about any areas where improvement is needed as well as areas where improvements have been made.


The “View learning materials” option will proceed to ask the student what school year, subject and topic within the subject they would like learning materials for. The application will display a list of learning material documents stored within the system as well as links to external learning material on the web.


The “view profile” option will display all the results for each quiz taken and also the improvements made during the year. A ranking of all registered users will also be displayed for each year and subject.

Once the user is logged in as a parent a menu will appear with two options:


  • View linked student accounts,
  • View Parent forum.

The “View Linked student accounts” option will display a list of student account that you have linked to your account. You can select a student account and review progress reports, results and rankings. If no student accounts are linked the application will display an appropriate message.

The “View parents forum” will display a list of discussions available for each year and subject.

Storage of all the quiz questions will be in a file within the system and user database will be located in a MySQL database using phpMyAdmin.
