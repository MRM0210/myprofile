Project Idea

Overview

The project that I have decided on is a smartphone and desktop learning web application for primary school aged students and parents. The application content will be based on the curriculum of each school year, and broken down into all the main subjects that students undertake during each year. Each subject will have quizzes that will test the child’s knowledge and report back on areas they need to improve on. Each students progress will be ranked against all other students in the system for each school year and subject. All information will be accessible by registered parents. There will also be a discussion forum for the parents to communicate with each other about the curriculum and share teaching techniques.

Motivation

The challenge of getting my two primary school aged children to complete their homework each night is the motivating factor behind this idea, to say it’s a challenge is a major understatement!!! The idea of having a fun and interactive application for my children to study relevant content at the click of a button while receiving instant feedback on their work is the driving force for this project. The Idea of a platform to communicate with other parents about the curriculum and share teaching techniques is also very appealing. I am also highly motivated by the challenge of programming an application like this using and learning the different languages and technologies required to complete this task.

Description

The application will start by asking the user to either register as a parent or student. The registration process for a student will ask the student for their country, state, name, age, gender and school year. The registration for a parent will ask for name, how many primary school aged children they have and weather they would like to link their account to a child’s account. The data provided will then be stored in a user data base.

Once the user is logged in as a student a menu will appear with the following options:

  • Take a quiz
  • View learning materials
  • View profile

The “Take a quiz” option will proceed to ask the student for a subject and a topic within the subject they would like for the quiz. After all the input is taken the application will proceed with a 20 question quiz on the subject topic and difficulty level of the school year selected on their profile, all question will be relevant to the school curriculum of the state of the student. All results for each subject will be stored in a database and the student will be notified about any areas where improvement is needed as well as areas where improvements have been made.

The “View learning materials” option will proceed to ask the student what school year, subject and topic within the subject they would like learning materials for. The application will display a list of learning material documents stored within the system as well as links to external learning material on the web.

The “view profile” option will display all the results for each quiz taken and also the improvements made during the year. A ranking of all registered users will also be displayed for each year and subject.

Once the user is logged in as a parent a menu will appear with two options:

  • View linked student accounts,
  • View Parent forum.

The “View linked student accounts” option will display a list of student accounts that you have linked to your account. You can select a student account and review progress reports, results and rankings. If no student accounts are linked the application will display an appropriate message.

The “View parents forum” will display a list of discussions available for each year and subject. This feature will give parents a chance to talk to other parents about the curriculum of other schools and states.

The application will have a user interface that is simple for the user to understand yet still be aesthetically pleasing while at the same time having all the relevant features that are required.

A large data base of questions will be stored in the system and the questions will be selected at random when quizes are taken. This will ensure that each quiz taken will be different.

Tools and Technologies

Some of the tools and technologies required include languages such as HTML, CSS and JavaScript for the front end development and user interface, while programming languages such as Java, PHP, Python or Ruby are all solid choices for the back end development tasks such as input validation and login systems. There are also many frameworks available to choose from such as Django for python, Laravel for PHP and Ruby on Rails which simplifies the task of developing a web application, making the task a lot faster and easier by doing a lot of the back end work for you. There are also many choices for databases handling such as phpMyAdmin which is a free and open source tool for MySQL.

The main skills needed to complete this project are:

  • The ability to design and develop a user interface that is simple for the user to understand yet still be aesthetically pleasing while at the same time having all the relevant features that are required.
  • The ability to write program logic in one of the above mentioned programming languages and/or frameworks to solve problems efficiently and to develop the application to a professional standard.
  • The ability to connect the application to a database to store all user data reliably and secure.

The skills required are all basic programming core skills that are very achievable to attain. There is hundreds of websites, forums and books online with up to date information on all the technologies needed for this project.

Outcome

The intended outcome of this application is to give students another option for learning the topics covered in school. The objective of the project is to create a fun and interactive learning environment for students while hopefully having a positive impact on the grades achieved by students. Keeping users informed on how well they are going in relation to other students of the same age may result in the child working harder to move up the rankings. Parents being able to communicate with other parents about anything related to the curriculum may result in resources and teaching techniques being shared around, which can only result in a positive outcome for both students and parents.
